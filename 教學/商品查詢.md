## 內期商品查詢

透過 [get_domestic_products]({{ "/API/unitrade/#get_domestic_products" | relative_url }}) 方法可取得所有內期商品資訊（如商品代碼、合約規格等）。

```python
# 內期商品取得範例 
domestic_products = unitrade.get_domestic_products()
# print(domestic_products)
```

<details markdown="1">
<summary>點擊查看回傳資料結構範例</summary>

回傳值為一個 Dictionary，Key 為商品代碼。

```python
{
    'TXF': DomesticProduct(
        kinid='TXF',
        name='臺指',
        stock_id='',
        subtype='I',
        contract_size=200,
        strike_price_decimal_locator=0
    ), 
    # ......
} 
```
</details>

## 內期商品合約查詢 

透過 [get_domestic_contracts]({{ "/API/unitrade/#get_domestic_contracts" | relative_url }}) 方法查詢指定商品的合約列表（月份、履約價等）。
需傳入 **商品代碼** 與 **類別碼**，類別碼請參考 [內期商品類別]({{ "/說明/內期商品類別" | relative_url }})。

### 期貨合約

```python 
# 取得臺指期 (TXF) 期貨 (F) 合約
query_response = unitrade.get_domestic_contracts("TXF", "F")
```

<details markdown="1">
<summary>點擊查看 DomesticContractResponse (期貨) 範例</summary>

```python
DomesticContractResponse(
    ok=True, 
    error='', 
    data=[
        DomesticContract(
            prod_id='TXFG5',
            month='202507',
            cp=None,           # 期貨無買賣權
            stikeprice=0,      # 期貨履約價為 0
            maxprice=23815,
            minprice=19485,
            premium=21650,
            divdate='20251217'
        ),  
        # ......
    ]
)
```
</details>

### 選擇權合約

```python
# 取得臺指選 (TXO) 選擇權 (O) 合約
query_response = unitrade.get_domestic_contracts("TXO", "O")
```

<details markdown="1">
<summary>點擊查看 DomesticContractResponse (選擇權) 範例</summary>

```python
DomesticContractResponse(
    ok=True, 
    error='', 
    data=[
        DomesticContract(
            prod_id='TXO15900G5',
            month='202507',
            cp='C',            # C: Call / P: Put
            stikeprice=15900,  # 履約價
            maxprice=7970,
            minprice=3570,
            premium=5770,
            divdate='20251217'
        ),
        # ......
    ]
)
```
</details>

## 外期交易所查詢

透過 [get_exchanges]({{ "/API/unitrade/#get_exchanges" | relative_url }}) 查詢系統支援的外期交易所資訊。

```python
# 外期交易所取得範例 
exchanges = unitrade.get_exchanges()
```

<details markdown="1">
<summary>點擊查看回傳資料結構範例</summary>

回傳值為 Dictionary，Key 為交易所代碼。

```python
{
    'CME': EXCHANGE(
        exchange='CME',
        name='芝加哥商品交易所',
        country='USA',
        currency='USD',
        shortname='CME'
    ),
    # ...... 
} 
```
</details>

## 外期商品查詢

透過 [get_foreign_products]({{ "/API/unitrade#get_foreign_products" | relative_url }}) 查詢外期商品清單。

```python
# 外期商品取得範例 
foreign_products = unitrade.get_foreign_products()
```

<details markdown="1">
<summary>點擊查看回傳資料結構範例</summary>

回傳值為 Dictionary，Key 為 `交易所|商品代碼`。

```python
{
    'CBT|10Y': ForeignProduct(
        exchange='CBT',
        symbol='10Y',
        type='F',
        name='微型 10 年收益率期貨',
        shortname='微型十年收益',
        country='USA',
        currency='USD'
    ), 
    # ...... 
} 
```
</details>

## 外期商品合約查詢

透過 [get_foreign_contracts]({{ "/API/unitrade#get_foreign_contracts" | relative_url }}) 查詢外期商品合約詳細資料。
需傳入 **交易所**、**商品代碼** 及 **商品類型**，代碼請參考 [外期商品]({{ "/說明/外期商品" | relative_url }})。

### 外期期貨合約

```python
# 查詢 CME 交易所, NQ 商品, 期貨 (F) 合約
response = unitrade.get_foreign_contracts("CME", "NQ", "F")
```
