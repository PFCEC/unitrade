## API 錯誤代碼表

以下列出 SDK 回傳的錯誤代碼 (`MSGxxx`) 及其代表意義。這些錯誤通常與參數輸入不正確有關。

| 代碼 | 描述與處理建議 |
| :--- | :--- |
| **MSG000** | 買賣別輸入錯誤！請輸入 `B` (買進) 或 `S` (賣出)。 |
| **MSG001** | 委託種類輸入錯誤！請輸入 `R` (ROD), `F` (FOK), `I` (IOC)。 |
| **MSG002** | 下單種類輸入錯誤！請檢查相關旗標參數。 |
| **MSG003** | 沖銷輸入錯誤！請輸入 `Y` 或 `N`。 |
| **MSG004** | 格式錯誤。 |
| **MSG005** | 必填欄位缺漏！ |
| **MSG007** | 超過委託筆數限制。 |
| **MSG008** | 請輸入有效委託書號。 |
| **MSG009** | 數值必須大於 0。 |
| **MSG010** | 超過單筆下單口數限制。 |
| **MSG011** | Call/Put 輸入錯誤！期貨請留空，Call 輸入 `C`，Put 輸入 `P`。 |
| **MSG012** | 履約價請輸入數值格式。 |
| **MSG013** | 下單價格別錯誤！請輸入 `M`(市價), `L`(限價), `3`(停損市價), `4`(停損限價)。 |
| **MSG014** | 新平倉碼輸入錯誤！請輸入 `Y`(新倉) 或 `N`(平倉)。 |
| **MSG015** | 當沖碼輸入錯誤！請輸入 `Y`(當沖) 或 `N`(非當沖)。 |
| **MSG016** | 尚未連線，請確認網路或 API 連線狀態。 |
