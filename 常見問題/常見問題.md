# 常見問題與錯誤排除

本章節彙整了在使用 SDK 過程中可能遇到的錯誤訊息與排除方法，分為以下類別：

* **[登入失敗]({{ "/常見問題/登入失敗/" | relative_url }})**：包含 Python 環境錯誤、帳號密碼錯誤、憑證錯誤等。
* **[下單失敗]({{ "/常見問題/下單失敗/" | relative_url }})**：包含權限不足、連線中斷、憑證檢核失敗等。

若您遇到的錯誤未列於此，請確認錯誤代碼並聯絡客服人員。
